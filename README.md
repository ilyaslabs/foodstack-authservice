# Configurations

## Auth-client

```yaml
auth:
  client:
    tokenUrl: http://localhost:8080/api/v1/authenticate
    refreshTokenUrl: http://localhost:8080/api/v1/refresh
    username: system # username/email of the user with SYSTEM scope
    password: system # password of the user with SYSTEM scope
```

## Auth-service-client

```yaml
auth:
  baseUrl: http://localhost:8080
```
